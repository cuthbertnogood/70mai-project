# Каталог "анализ" — автопилот 70mai

Сюда сохраняются структурированные описания, планы и результаты анализа логов, производительности и процесса обработки видео с SD-карт.

## Содержимое

- `план_анализа_логов_и_максимальной_производительности.md` — полный план аналитики по шагам (копирование, merge, compose, upload). Содержит:
  - Источники данных (publish_all.log + исторические import-логи)
  - Разделение по периодам / "SD-картам"
  - Детальный разбор каждого шага
  - Ключевые параметры (PROBE_WORKERS=8, MERGE_WORKERS=1 и др.)
  - Методика поиска максимальной производительности
  - Рекомендации по реализации парсера

## Будущие файлы (предлагается)

- `analyze_logs.py` — скрипт парсинга логов + таблицы throughput
- `best_configurations.md` — результаты анализа (лучшие конфиги по шагам)
- `performance_report_YYYY-MM.md` — отчёты по конкретным периодам
- CSV / таблицы с метриками

## Как использовать

1. Добавляй новые логи/запуски — обновляй анализ.
2. После каждого большого прогона сохраняй сюда выводы.
3. Ссылайся на этот каталог при обсуждении производительности.
